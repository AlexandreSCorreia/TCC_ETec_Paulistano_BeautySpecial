BeautySpecial

Sistema de controle de salão de beleza.

O software BeautySpecial foi criado para o controle e gerenciamento do administrador e funcionário, tendo em vista alguns recursos como gerenciamento e cadastro de cliente, cadastro de usuários, marcação de hora, controle financeiro e entre outros.

  • Data: Junho 2018
  • Cliente: Redanyas
  • Categoria: Sistema Desktop

BeautySpecial Mobile

Aplicativo de agendamento de salão de beleza.

Com o intuito de aumentar a facilidade dos clientes, nós da Easy Way projetamos o aplicativo mobile do sistema BeautySpecial. Ainda em fase de aperfeiçoamento e desenvolvimento de atualizações, o aplicativo proporciona uma interface simplificada, para a facilidade do cliente, trazendo o agendemento e cadastro do cliente, podendo também atualizar as informações pessoias do cliente e consultar o status do agendamento.

  • Data: Novembro 2018
  • Cliente: Redanyas
  • Categoria: Sistema Mobile
